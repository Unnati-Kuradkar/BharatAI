🤖 BharatAI

AI Based Government Scheme Recommendation System

BharatAI is an intelligent platform that helps citizens discover government schemes, scholarships, loans, and opportunities based on their personal profile.

Features

  • User profile-based recommendations
  • Government scheme search
  • AI assistant for scheme information
  • Eligibility checking system
  • Personalized opportunities

Technology Used

  • Python
  • Flask Framework
  • SQLite Database
  • HTML & CSS
  • AI Recommendation Engine
